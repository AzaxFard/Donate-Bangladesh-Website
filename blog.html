What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each part of the document (such as elements, attributes, and text) is a node in the tree. Through the DOM, programming languages like JavaScript can dynamically interact with, modify, and update the content, structure, and style of a webpage.

How do you select and element from the DOM?

You can select an element from the DOM (Document Object Model) using various JavaScript methods. The most common methods are:
1.getElementById(): Selects an element by its id attribute.
2.getElementsByClassName(): Selects all elements with a specific class name. Returns a collection (HTMLCollection).
3.getElementsByTagName(): Selects all elements with a specific tag name (e.g., div, p). Also returns a collection.
4.querySelector(): Selects the first element that matches a specified CSS selector.
5.querySelectorAll(): Selects all elements that match a specified CSS selector. Returns a NodeList.
These methods provide flexible ways to access and manipulate specific parts of a webpage.

What is event delegation in the context of the DOM?

Event delegation is a technique in JavaScript for handling events at a higher level in the DOM rather than attaching event listeners to individual elements. Instead of adding event listeners to multiple child elements, you attach a single event listener to a parent element. This listener then captures events that bubble up from the child elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using the DOM in JavaScript as follows:
1. Manipulating Attributes
setAttribute(): Adds or changes an attribute.
getAttribute(): Retrieves the value of an attribute.
removeAttribute(): Removes an attribute.
2. Manipulating Styles
style Property: Directly changes an element's inline CSS styles.
classList.add(): Adds a class to an element.
classList.remove(): Removes a class from an element.
These methods allow you to dynamically modify HTML elements and their styles directly from JavaScript
